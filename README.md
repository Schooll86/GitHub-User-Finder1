# GitHub-User-Finder1

## Автор
Обучающийся

## Описание
Приложение для поиска пользователей GitHub по имени и отображения информации о профиле.

## Функции
- Поиск пользователей GitHub
- Отображение информации о профиле (логин, аватар, репозитории)
- Просмотр списка найденных пользователей
- Работа с GitHub API

## Требования
- Python 3.x
- библиотека requests

## Запуск
1. Установить Python
2. Установить зависимости:

   pip install requests

3. Запустить:

   python main.py
